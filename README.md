# Cash Flow
this is a webpage where you can keep track of monthly expenses, and see how in which month you had the most expenses throughout a filtered year.

## Built With
Technologies
 - JavaScript
 - HTML/CSS
 - React.js
 
# Live Demo
[Demo](https://mhamad-raad.github.io/React-Cash-Flow/)
